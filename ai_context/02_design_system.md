# 02 Design System

## Theme
- **Style**: Material 3 Design
- **Mode**: System (Light/Dark support required)

## Colors (TBD - Placeholders)
- **Primary**: Deep Purple (Seed)
- **Background**: Standard scaffolding
- **Error**: Standard Red

## Typography
- **Font Family**: Default (San Francisco / Roboto) for now. Ideally 'Inter' or 'Poppins'.

## UI Rules
- Use `const` constructors wherever possible.
- Extract widgets if they exceed 50 lines or are reused.
- Padding: Use multiples of 4 or 8 (e.g., 8.0, 16.0, 24.0).
- BorderRadius: Consistent 12.0 or 16.0 for cards/buttons.
